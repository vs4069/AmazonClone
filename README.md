# AmazonClone


![AmazonClone ](/demoImages/amazonclone0.png)

## Description

This is a responsive static website that serves as a clone of Amazon, an e-commerce platform. The website replicates the user interface and design of the Amazon platform using HTML and CSS for styling. It is not functional and does not include any JavaScript for interactivity. The website showcases the layout and responsive design of the Amazon platform on different devices.
